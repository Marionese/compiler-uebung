Datenstrukturen
Hier folgt die Aufzählung aller Datenstrukturen mit einer Kurzbeschreibung:
 CAssignEine Zuweisung: lhs = rhs
 CBinOpExprEine binäre Operation
 CBlockEin Block von Anweisungen, der selbst eine Anweisung ist
 CDefIdEine eindeutige Kennung für die (Funktions- oder Variablen-)Definition, auf die ein auflösbarer Bezeichner (ResIdent) verweist
 CDefInfoVariantentyp für semantische Informationen
 CDictWörterbuch
 CDictEntryWörterbucheintrag
 CExprEin Ausdruck
 CForInitDer Initialisierungsparameter einer For-Anweisung
 CForStmtEine For-Anweisung: for (init; cond; update) body
 CFuncCallEin Funktionsaufruf
 CFuncDefEine Funktionsdefinition
 CFuncInfoStruktur für semantische Informationen über Funktionen
 CFuncParamEin Funktionsparameter mit einem Typ und einem (nicht auflösbaren) Namen
 CIfStmtEine If-Anweisung: if (cond) if_true else if_false
 CItemEin Top-Level-Programmelement: Die Definition einer globalen Variable oder Funktion
 CItemIdEine eindeutige Kennung für die Top-Level Elemente (Item) des abstrakten Syntaxbaumes (Program)
 CLiteralEin Literal-Ausdruck, z.B
 CPrintStmtEine print-Anweisung, die zur Ausgabe von Strings oder Ausdrücken genutzt wird
 CProgramDer oberste Knoten des abstrakten Syntaxbaums (AST) für ein Programm
 CResIdentEin auflösbarer Bezeichner, der auf eine (Funktions- oder Variablen-) Definition verweist
 CStmtEine Anweisung
 CSymDefTableStruktur mit semantischen Informationen, die in Kombination mit dem abstrakten Syntaxbaum für die Interpretation und weitere Verarbeitung nützlich ist
 CSymtabStruktur der Symboltabelle
 CSymtabSymbolStruktur eines Symbols in der Symboltabelle
 CVarDefEine Variablendefinition
 CVarInfoStruktur für semantische Informationen über Variablen
 CVecHeaderStruktur, die dem Array zur Buchhaltung vorangestellt ist
 CWhileStmtEine While- oder Do-While-Anweisung: while (cond) body / do body while (cond)
